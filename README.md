## ğŸ’» Tech Stack

This project is a classic MERN (MongoDB, Express, React, Node.js) stack application, demonstrating proficiency across the entire development spectrum.

### Frontend
* **React:** Component-based UI development.
* **Axios:** Handling all API communication.
* **CSS Grid/Flexbox:** Professional, responsive styling.

### Backend & Database
* **Node.js / Express:** RESTful API server.
* **MongoDB / Mongoose:** Non-relational database for recipe storage and schema definition.
* **External API:** [Spoonacular](https://spoonacular.com/food-api) (used for recipe search and data).

### Deployment
* **Vercel:** Hosting the static React client.
* **Render:** Hosting the Node.js API server.
* **MongoDB Atlas:** Cloud-based database hosting.
